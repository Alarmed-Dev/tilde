Empty categories

Emptying a category will remove all of the notes associated with the given category. Note that every possible category and section already exists. You simply need to add notes to them.

Empty sections


Emptying a section will remove all of the notes associated with the given section. Note that every possible category and section already exists. You simply need to add notes to them.
